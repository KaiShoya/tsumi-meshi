# v0.1 タスク（最新まとめ）

このファイルは v0.1 系のマスタータスクリストです。既にリリースされた v0.1.1〜v0.1.3 の完了項目はアーカイブ済み、残タスクは短期リリース（v0.1.4 / v0.1.5）へ振り分けています。詳細は `/.agent/docs/tasks/v0.1.4-tasks.md` と `/.agent/docs/tasks/v0.1.5-tasks.md` を参照してください。

## 完了済み（アーカイブされました）
- Nuxt 4 プロジェクト（TypeScript）
- NuxtUI と Tailwind CSS の設定
- 認証システム（Cloudflare Workers + JWT）
- Cloudflare D1 スキーマ
- リポジトリクラス（Recipes, Folders, Tags 等）
- Pinia ストア（data / pages）
- レシピ CRUD UI（一覧・作成・編集・削除）
- タグ管理（API / UI / store）
- チェック機能（履歴・統計の基礎）
- 検索・フィルタ（バックエンドとフロントの統合）
- 初期テスト基盤（Vitest + setup）
